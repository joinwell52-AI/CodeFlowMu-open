# OPS Log Diagnosis

Use this playbook when OPS must summarize logs into actionable evidence.

## Output

```text
# Log Diagnosis

## 1. Log Sources
Files, commands, time range.

## 2. Signals
Errors, warnings, repeated events, stalls.

## 3. Timeline
Short event order.

## 4. Likely Cause
Evidence-backed hypothesis.

## 5. Next Action
Owner and action.
```

## Rules

- Quote only small relevant excerpts.
- Do not paste full private logs into public drafts.
- Distinguish evidence from inference.
